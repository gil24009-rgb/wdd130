Overview

Purpose

Photo Gallery is a simple, visually focused website where visitors can browse curated photo collections without distractions. The goal is to present images in a clean, readable layout that works well on both mobile and desktop while providing clear navigation and a consistent visual style.

Audience

The primary audience is general visitors who enjoy photography and want an easy browsing experience. Secondary audiences include classmates and instructors reviewing layout, responsiveness, and visual design quality in a web project.

Typography

Headings use a clean display font for strong hierarchy, and paragraphs use a highly readable sans serif for comfortable scanning on screens. This pairing supports a gallery style site where images are the main focus and text stays clear and minimal.

Content

Home page

Welcome to Photo Gallery, a place to explore curated image collections organized by theme. Use the Gallery page to browse sets like landscapes, textures, and city scenes, then open any image to view it larger with its caption. If you want to understand the goal of the site and how the collections are organized, visit the About page for a short overview.

Images for the Home page

Gallery

The Gallery page presents multiple image collections in a grid layout that adapts to screen size. Each image includes a short caption to provide context. Visitors can select any image to view a larger version and then return to the grid without losing their place, making browsing smooth and predictable.

Images for the Gallery page

About

Photo Gallery is designed around simple navigation, strong readability, and consistent spacing so images remain the focal point. Collections are grouped by theme and use clear captions to reduce confusion while browsing. The overall design choices support an enjoyable viewing experience on both mobile and desktop without requiring accounts, external data, or complex interactions.

Images for the About page

Wireframes

Create three wireframes for your site, one for each page, and list them here.

Home

The home page wireframe should show a hero image, a short welcome paragraph, and two featured collection tiles that link to the Gallery. Navigation stays at the top for quick access on all pages.

Gallery

The gallery wireframe should show a responsive grid of images with captions. Selecting an image opens a larger view with a close control, then returns the visitor to the same scroll position in the grid.

About

The about wireframe should show a short site description, a small section explaining how collections are organized, and a simple contact line. One supporting image should appear near the top to keep the page visually consistent with the rest of the site.
